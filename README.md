Mess of code for portfolio. Created by me guided by AI.
Multiplayer game. 
U need launch `go run .` in main_server directory -> to start server
And `go run .` in main_client directory -> connect to server
.env file for server represent listen port and what ip to listen ( change to 0.0.0.0 to listen all incoming request )
.env for client represent server port and ip address where connect ( change to port from .env server and ip of the machine where launch server )
